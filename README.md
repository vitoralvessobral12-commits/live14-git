# 🍔 Cardápio Digital Online DC-L13

Um projeto didático criado para explorar e aplicar conceitos fundamentais de **HTML5** e **CSS3** no desenvolvimento de interfaces web responsivas.

## 📚 Contexto do Projeto

Este projeto foi desenvolvido como parte do **Módulo de HTML e CSS** do curso de **Desenvolvimento Fullstack**, servindo como base prática para consolidar conhecimentos de:
- Estruturação semântica de documentos HTML
- Estilização com CSS moderno
- Design responsivo e UX principles
- Organização de assets e recursos

---

## 🛠️ Tecnologias Utilizadas

### **HTML5**
- **Impacto:** Fornece uma estrutura semântica robusta com tags apropriadas (`<header>`, `<nav>`, `<h1>-<h2>`), melhorando acessibilidade e SEO. A linguagem de marcação padroniza a organização do conteúdo.
- **Uso no projeto:** Define a estrutura de navegação, catálogo de produtos e elementos interativos.

### **CSS3**
- **Impacto:** Proporciona estilos visuais profissionais e consistentes. Utiliza propriedades modernas como `flexbox` para layouts flexíveis, `border-radius` para design arredondado, e `rgba()` para transparência sofisticada.
- **Uso no projeto:** Cria cards de produtos, distribui elementos com flexbox, aplica paleta de cores unificada (#1e1f22, #FFAE00).

### **Flexbox (CSS3 Layout)**
- **Impacto:** Permite alinhamento e distribuição inteligente de elementos sem necessidade de floats ou positioning absoluto. Facilita criação de layouts responsivos economicamente.
- **Uso no projeto:** 
  - Distribuição horizontal/vertical de itens nos cards
  - Centralizando a navegação no header
  - Garantindo espaçamento uniforme entre componentes

### **Responsive Design** (Mobile-First)
- **Impacto:** A aplicação funciona perfeitamente em diferentes tamanhos de tela (desktop, tablet, mobile), aumentando alcance e usabilidade da aplicação.
- **Uso no projeto:** Viewport meta tag configurada, layouts fluidos, imagens escaláveis.

### **Imagens Otimizadas**
- **Impacto:** Pasta `imagensCompactadas/` sugere processamento de imagens para reduzir tamanho de arquivo, melhorando performance de carregamento e experiência do usuário.
- **Uso no projeto:** Logo, fotografias de produtos (burgers e bebidas), background da página.

### **Paleta de Cores Profissional**
- **Impacto:** Decisão de design estratégica com cores coerentes cria identidade visual forte:
  - **#1e1f22** (Preto): Sofisticação e foco
  - **#FFAE00** (Laranja): Destaque e call-to-action
  - **RGBA(37, 37, 37, 0.8)** (Cinza translúcido): Contraste com transparência elegante
- **Uso no projeto:** Cria hierarquia visual clara e profissionalismo.
